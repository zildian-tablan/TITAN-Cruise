# Titan Cruise Booking System

An Java academic project that demonstrates a simple cruise booking system. Users can register, log in, browse cruises, make bookings, and print receipts. An admin interface lets instructors or students view and manage bookings. This project is intended for learning and demonstration, not production.

## About / Purpose

This academic project demonstrates how a basic booking system works and teaches core concepts, with learning goals including account management, booking flows, simple GUI interaction, and basic data handling.

## Key components
- User interface: simple screens for signup, login, browsing cruises and managing user profiles. Forms guide the user and show confirmation or error messages.
- Booking flow: choose a cruise, select dates and , enter passenger details, review the summary, and confirm the booking.
- Receipts: a booking confirmation view that shows trip details, passenger info, and price; can be saved for records.
- Admin: basic instructor/demo features to list and inspect bookings and users, cancel or mark bookings as handled, and view summaries for testing.

## Help 

- To run: build with Maven and run `com.mycompany.titanproj.TitanProj`.
- If the UI doesn't show, check you have Java installed and run from the project root.
- If images are missing, ensure the `titanimages/` folder is in the working directory when launching the app.
 
## Project structure

- `pom.xml` - Maven project file and dependencies.
- `src/` - project source and resources.
- `target/` - Maven build output.
- `titanimages/` - Images used by the application.
